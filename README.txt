
1. Open the Project: Open the Emre_Ercan_HW1 folder in NetBeans.

2. Run it: Right-click the Emre_Ercan_HW1.java file (Located at Emre_Ercan_HW1/src/emre_ercan_hw1) and select Run File.

3. All input and output text files are created in the project's root directory (Emre_Ercan_HW1/).

All auxiliary files used for graphical analysis are located in the analysis_tools/ folder.
